================================================================================
                            WIDGET API METHODS
================================================================================

Initialize:
  new ChatWidget({
    chatbotId: 'bot-123',
    apiBaseUrl: 'http://localhost:8080/v1/api/n8n/multimodal'
  })

Send Message:
  await chat.sendMessage({
    message: 'Hello!',
    attachments: [file1, file2]  // Optional
  })

Upload File:
  await chat.uploadFile(file)

List Attachments:
  await chat.listAttachments()

Delete Attachment:
  await chat.deleteAttachment(vectorId)

Get Session ID:
  chat.getSessionId()

Get Chatbot ID:
  chat.getChatbotId()

================================================================================
                        TROUBLESHOOTING GUIDE
================================================================================

Issue: "Cannot find ChatWidget"
Fix: Ensure dist/index.js is loaded before using

Issue: "Failed to fetch from API"
Fix: Check backend is running, verify apiBaseUrl

Issue: "401 Unauthorized"
Fix: Verify chatbotId and sessionId are correct

Issue: "File upload fails"
Fix: Use endpoint http://localhost:8080/api/attachments/upload
     Include chatbotId and sessionId in form data

Issue: CORS errors
Fix: Backend CORS should be configured for your frontend domain
